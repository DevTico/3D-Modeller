Geometria 3D

Editor 3D interativo com GeoScript, biblioteca de objetos, animações procedurais e muito mais. Rode direto no browser — só precisa do three.min.js na mesma pasta.

O que você pode fazer

✦ Criar e editar objetos 3D  ✦ Animar com GeoScript  ✦ Sistemas procedurais (solar, cidade, asteroides)  ✦ Biblioteca com 35+ objetos  ✦ Gradientes, imagens de fundo  ✦ Câmera livre com orbit e WASD

Início rápido
1. Clique em ⊞ para abrir a ToolBox e adicionar formas 2. Clique em 🧱 Biblioteca para objetos prontos 3. Use ⚙ Props para ajustar posição, escala e cor 4. Abra ✦ GeoScript e experimente um exemplo 5. Pressione ▶ Executar para rodar o script

GeoScript — Básico

GeoScript é a linguagem de animação e criação do editor. Cada linha é um comando. Linhas começando com // são comentários.

Spawn
spawn box at 0,0,0 color #ff6600 scale 1.2 spawn sphere at -2,0,1 color #5c9aff scale 0.8
Cor e Fundo
color #ff0000 target 0 background #020820 grid off grid size 40
Formas disponíveis

box sphere cone cylinder torus octahedron heart text3d minimal claude arrow knot star tetrahedron dodecahedron capsule ring

Variáveis e Matemática
set x to 0 set cor to #ff6600 set qtd to 5 set x to x + 1.5 set resultado to x * 2 + 1

Variáveis funcionam em todos os parâmetros: posição, cor, escala, contagem de loops, etc.

set raio to 3 spawn sphere at raio,0,0 scale 0.5 spawn sphere at -raio,0,0 scale 0.5
Loops
Repeat
set x to -3 repeat 7 spawn sphere at x,0,0 color #8c6a4e scale 0.5 set x to x + 1 end
For
for i from 0 to 11 spawn minimal at 0,0,0 scale 0.4 end animate orbit radius 3 speed 0.8
For com step
for angle from 0 to 360 step 45 spawn box at 0,0,0 scale 0.5 end
Animações

Use animate <efeito> [parâmetros] para adicionar animações. Múltiplas animações podem ser combinadas.

EfeitoParâmetrosDescrição
spinaxis speedRotação contínua
floatamount speedFlutuar para cima/baixo

color_cyclespeedCiclo de cores HSL
pulsespeed amountPulsar escala
orbitradius speed axisÓrbita circular
gravitystrengthAtração gravitacional
magnetstrength range targetAtrair objetos
traillengthRastro atrás do objeto

glowspeed amountBrilho pulsante

Exemplo combinado
spawn sphere at 0,0,0 color #7c9aff scale 1.2 animate spin axis y speed 1 animate float amount 0.4 speed 0.8 animate glow speed 1.5 amount 0.5 animate particle_emitter rate 8 color #aaccff

Biblioteca de Objetos

Clique em 🧱 Biblioteca para abrir. Cada categoria tem objetos pré-configurados com animações e materiais.

Como usar

Clique em qualquer objeto para adicionar na cena. Arraste o objeto diretamente para a cena para colocar onde quiser. Clique em ⭐ para favoritar — favoritos ficam salvos no browser.
